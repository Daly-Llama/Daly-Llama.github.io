# Dallin Nielson — Data Science Portfolio

**Live site:** [daly-llama.github.io](https://daly-llama.github.io)

This repository contains the source for my data science portfolio site, built with Jekyll and hosted on GitHub Pages. The site showcases applied ML and NLP projects focused on knowledge systems, support operations, and human-centered AI.

---

## About the Site

The portfolio is organized around my core domain: **Knowledge Management & Support Operations** — projects that apply machine learning to real knowledge workflows, built around the KCS (Knowledge-Centered Service) methodology. Additional projects are included as well.

---

## Repository Structure

```
_layouts/
  default.html        # Global layout: nav, header, footer
assets/
  css/
    style.css         # All styling via CSS custom properties
  *.jpg / *.png       # Site images and icons
index.md              # Home page
about.md              # Bio, values, skills
projects.md           # Project portfolio
contact.md            # Contact information
_config.yml           # Site metadata (title, links, description)
```

---

## Built With

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting
- [Google Fonts](https://fonts.google.com/) — Playfair Display, Source Sans 3, JetBrains Mono
- Custom CSS (no frameworks)
